# AviationDB_CSharp
О проекте AviationDB_CSharp — это зачетный проект по предмету "ADO.NET", представляющий собой desktop-приложение для управления базой данных авиарейсов, разработанное на C# с использованием:

- Архитектуры MVVM для четкого разделения ответственности;
- Полного набора CRUD-операций для работы с данными;
- Графического интерфейса на WPF Application;
- Технологии EF для взаимодействия с базой данных PostgreSQL.

### Основные возможности Создание, просмотр, редактирование и удаление записей о авиарейсах
- Поиск данных;
- Пагинация данных - постраничное отображение записей для удобной работы с большими объемами информации
- Валидация вводимой информации;
- Интуитивно понятный графический интерфейс;
